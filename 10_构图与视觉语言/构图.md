# 构图

定义：根据题材、主体、目的，把元素组成协调的画面

要点

1.构图服务于目的



|    类别    |     名称     |       描述       |                             备注                             |
| :--------: | :----------: | :--------------: | :----------------------------------------------------------: |
|   点构图   |   中心构图   | 把主体放到中心点 |                                                              |
|   点构图   |  三分点构图  | 把主体放到三分点 |                                                              |
| 对称线构图 | 水平对称构图 |                  |                             水面                             |
| 对称线构图 | 垂直对称构图 |                  |                             建筑                             |
| 三分线构图 |              |                  | 强调的主体占据更大面积<br />人像要注意视觉引导，人看向更长的画面方向 |
| 对角线构图 |   S线构图    |                  |                                                              |
| 引导线构图 |              |                  | 线条引导到主体上，容易突出主体，有层次感<br />最好主体在三分点或者中心点上 |
|   面构图   |   前景构图   |                  |            框架构图属于前景构图，可以突出画面主体            |
|   面构图   |   填充构图   |                  |                      夸张的人物或者动物                      |
|   面构图   |   留白构图   |                  |                  和填充构图相反，小清新风格                  |
|   面构图   |   重复构图   |                  |                      基于规则的重复逻辑                      |
|   面构图   |   对比构图   |                  |                  和重复构图相反，突出差异性                  |
