# 镜头评估

## 如何评估一颗好镜头

|      类别      |      一级因素中文名      |               一级因素                |         二级因素中文名         |                             解释                             |
| :------------: | :----------------------: | :-----------------------------------: | :----------------------------: | :----------------------------------------------------------: |
|    光学质量    |     **锐度/分辨率**      |         Sharpness/Resolution          |                                | 评价镜头的分辨能力，通常以线对每毫米（线/mm）或像素来衡量。高分辨率意味着镜头能够捕捉更多的细节。 |
|    光学质量    |       锐度/分辨率        |         Sharpness/Resolution          |      **锐度(Sharpness)**       | 锐度是指图像或照片中细节的清晰度和边缘的清晰度。在摄影中，一个锐度高的图像具有清晰的细节和边缘，而一个锐度低的图像可能会显得模糊或缺乏细节。锐度通常是镜头和相机的性能因素之一，但也受到拍摄条件、焦距和后期处理等因素的影响。 |
|    光学质量    |       锐度/分辨率        |         Sharpness/Resolution          |     **分辨率(Resolution)**     | 分辨率是图像中可见细节的数量和清晰度的度量。通常以像素为单位来表示，更高的分辨率意味着图像具有更多的像素，可以呈现更多的细节。分辨率不仅受到摄像设备（例如相机和镜头）的影响，还受到图像传感器和输出媒体（例如打印或屏幕显示）的限制。<br />尽管锐度和分辨率都涉及图像的清晰度和细节，但它们关注的方面不同。锐度通常是指图像的视觉感知清晰度，而分辨率是一个数值度量，表示图像中可以捕捉的细节数量。在某些情况下，高分辨率图像可能具有更高的锐度，因为它们包含更多的细节，但两者并不总是直接相关。 |
|    光学质量    |         **畸变**         |              Distorsion               |                                |          畸变是指在摄影中直线看起来弯曲或变形的现象          |
|    光学质量    |           畸变           |              Distorsion               |          **桶形畸变**          | 这种畸变使图像中的直线向图像中心弯曲，形状呈现出向外凸出的桶形。 |
|    光学质量    |           畸变           |              Distorsion               |          **枕形畸变**          | 这种畸变使图像中的直线向图像中心收缩，形状呈现出向内凹陷的枕形。 |
|    光学质量    |         **色散**         |              Dispersion               |                                | 色散是光线在穿过透镜或折射体时，不同波长的光线因折射率不同而产生不同的弯曲程度。这会导致不同波长的光线在焦点处汇聚于不同的位置，从而引起彩虹效应。 |
|    光学质量    |           色散           |              Dispersion               | **色差(Chromatic Aberration)** | 色差是指光线在穿过透镜或透镜系统时，不同波长的光线会在不同的焦点处聚焦，导致图像中出现色彩偏移。 |
|    光学质量    |         **散射**         |                 Flare                 |                                |      评估在强光条件下，光线是否会散开，产生光晕或幻影。      |
|    光学质量    |         **暗角**         |              Vignetting               |                                | 暗角是指图像的角部相对于中心部分较暗的现象。这种现象通常在镜头的边缘产生，因为在图像中心部分使用了更多的光线，而在边缘则使用较少的光线。这种效果有时被摄影师用来增强中心主题的注意力，但也可以通过后期处理进行校正。 |
|    光学质量    |       **散焦模糊**       |                 Bokeh                 |                                | 评价镜头的背景虚化效果，以及在背景中的光斑的质量和外观。Bokeh 是一种特定的美学效果，通常是有意为之，以改善照片的外观。 |
|    光学质量    |       **色彩还原**       |            color rendition            |                                |   评估镜头的颜色还原能力，包括对真实颜色的准确性和饱和度。   |
| 光圈和快门性能 |         **光圈**         |            Aperture Value             |                                |         光圈的大小影响进光量和景深，通常以f/数表示。         |
| 光圈和快门性能 |       **快门速度**       |             Shutter Speed             |                                | 快门速度决定了曝光时间，对于决定拍摄运动或静止场景的清晰度至关重要。 |
|     稳定性     |         **防抖**         |          Image Stabilization          |                                | 防抖技术可以减少由于相机抖动而引起的图像模糊。这对于手持拍摄、低光条件下拍摄或使用较长焦距镜头拍摄时非常重要。相机可以采用机械式防抖、光学防抖或数字防抖来提高稳定性。 |
|     稳定性     |           三防           | waterproof, dustproof, and shockproof |      **防水(Waterproof)**      | 防水相机具有防水设计，使其能够在水下或湿润的条件下使用，例如水下摄影、雨中拍摄或在潮湿环境中拍摄。防水相机通常具有密封的外壳，防止水分渗透到相机内部，从而保护其电子元件。 |
|     稳定性     |           三防           | waterproof, dustproof, and shockproof |      **防尘(Dustproof)**       | 防尘相机具有密封设计，可以防止尘埃、灰尘和颗粒进入相机内部。这对于拍摄沙漠、沙尘暴或其他尘埃多的环境非常重要。防尘功能确保相机内部的镜头、传感器和其他关键组件保持清洁。<br />内变焦比外变焦更防尘。 |
|     稳定性     |           三防           | waterproof, dustproof, and shockproof |      **防震(Shockproof)**      | 防震相机具有强化的外壳和内部结构，可以抵抗摔落或冲击。这使得相机更耐用，可以承受不小心的碰撞或坠落，而不会受到严重的损害。 |
|     稳定性     |    **机械结构和材料**    |                                       |                                | 相机的外壳、内部结构和按键应采用坚固的材料，以提高耐用性和抗损坏性。高品质的材料和制造工艺可以使相机在使用中更加稳定。 |
|     稳定性     |      **温度稳定性**      |                                       |                                | 相机应具有一定的温度稳定性，可以在各种气温条件下正常工作。一些环境下的极端温度可能会影响相机的性能，因此温度稳定性是一个重要考虑因素。 |
|     稳定性     |       **镜片涂层**       |             Lens Coatings             |                                |   多层涂层可以减少反射和提高透明度。具有防尘、防油的效果。   |
